# ShadowNetClientLauncher

A simple web launcher for Shadow Net Client 

---

## 🚀 Features

- ShadowNetClient JavaScript 
- ShadowNetClient WASM 
- Quick access to other related clients
- Simple browser based launcher (Apex Launcher & Eclipse Client)

---

## 📂 Usage

Just open the site in a browser and click your version to launch.

**You are allowed to host this yourself just please dont take credit for making it**

If hosting locally:
- Open `index.html`
- Or host using GitHub Pages / any static web host
